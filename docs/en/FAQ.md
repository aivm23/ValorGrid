# FAQ

## Does ValorGrid send my portfolio anywhere?

No. Portfolio movements and the SQLite database remain local. Market-data lookups request configured symbols from the selected provider.

## Do I need Node.js?

Desktop releases include the runtime. Development and source execution require Node.js 24+.

## Can I import broker files?

Community imports only the official ValorGrid Excel template. Broker-specific importers are Professional/Enterprise functionality.

## Is this financial advice?

No. ValorGrid is a record-keeping, visualization and audit tool.
