# Create Instruments

Instruments define what ValorGrid can track.

Common fields:

- Symbol: local unique identifier.
- Provider reference: market-data symbol such as a Yahoo ticker.
- Name: display name.
- Type: stock, ETF, crypto, commodity or FX.
- Currency: instrument currency.
- Color: visual identity unless automatic brand palette is enabled.

Crypto and commodity positions use generic quantity wording in the UI. Stocks and ETFs may use share wording where appropriate.
