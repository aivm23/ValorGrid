# First Steps

Use this guide to get a clean ValorGrid Community installation running and add your first portfolio data.

1. Install the desktop app or start Docker/CasaOS.
2. Open ValorGrid locally.
3. Create instruments from the guided setup or the Instruments modal.
4. Add manual movements or import the official ValorGrid Excel template.
5. Review Operations, Allocation, YTD, History and Movements.
6. Create a backup before risky changes.

ValorGrid stores data in a local SQLite database. Market prices may be requested from the configured market-data provider, but portfolio movements and settings remain local.
