# Editions

## ValorGrid Community

Community is the public local-first portfolio app. It includes manual movements, the official ValorGrid Excel template import, dashboard, history, backups, Docker/CasaOS/Umbrel packaging and desktop builds.

## Professional / Enterprise

Professional/Enterprise editions may add private import adapters and advanced UI preferences.

Community may show public teasers such as "Professional Edition", but it must not expose private adapter ids, broker parser contracts, private fixture names, operational secrets or restricted workspace details.
