# /audit-docs - Auditoría de Documentación

Lee los archivos markdown de documentación y verifica referencias a rutas obsoletas.

**Comandos:**

- `npm run docs:spellcheck` — ortografía y acentos
- `node --test test/architecture.test.js` — tests de arquitectura

**Archivos:**

- `docs/*.md` — toda la documentación del proyecto
- `test/architecture.test.js` — cobertura arquitectónica
