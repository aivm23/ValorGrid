# Third-Party Notices

ValorGrid includes third-party open-source dependencies installed through npm. Those components remain subject to their own license terms as declared by their package metadata.

Primary dependency metadata is recorded in:

- `package-lock.json`
- `apps/server/package.json`
- `apps/desktop/package.json`
- `packages/contracts/package.json`

Release artifacts and Docker images may include production dependencies required to run ValorGrid. Review the package metadata for the complete dependency license set for a specific version.
